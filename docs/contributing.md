# Contributing

## Adding new prompts

1. Duplicate a prompt in `prompts/`.
2. Update it with app-specific details.
3. Reference the new prompt in `docs/architecture.md` if it introduces a new layer.

## Adding new templates

1. Create a new markdown file in `templates/`.
2. Keep sections consistent with existing templates.
3. Add examples that map to SAP Fiori patterns (lists, object pages, etc.).
